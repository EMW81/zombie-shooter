## What this game is

A 3D browser zombie shooter set in a blocky Roblox-style **Nuketown** map. You survive waves of zombies, collect money from kills, and buy better guns from a shop. Features horror synth music, Xbox controller support, and a 3-second get-ready countdown before each round.

## How to play

| Action | Keyboard/Mouse | Xbox Controller |
|---|---|---|
| Move | WASD | Left Stick |
| Aim | Mouse | Right Stick |
| Shoot | Left Click | RT (Right Trigger) |
| Open Shop | B | Menu Button |
| Switch Gun | 1–5 | RB / LB |

## The guns

1. **Pistol** — Free starter
2. **Shotgun** — $250, 6-pellet spread
3. **Assault Rifle** — $600, fast fire rate
4. **Sniper** — $900, piercing, high damage
5. **Minigun** — $2000, fastest fire rate

## The zombies

- 🟢 **Walker** — standard, balanced
- 🟠 **Runner** — fast, low HP (wave 3+)
- 🔴 **Tank** — slow, high HP, big money drop (wave 5+)

## Tech stack

- **Three.js r128** — 3D rendering
- **Vanilla JavaScript** — game logic
- **Web Audio API** — synthesized horror music (no files needed)
- **Gamepad API** — Xbox controller support
- **HTML/CSS** — UI overlays
